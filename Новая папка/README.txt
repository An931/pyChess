Игра Шахматы

Запуск через start_gui

Рекомендации по игре: 
перетаскивайте свои фигуры в соответствие с правилами, чтобы делать ходы

В файлах game, players, pieces находится логика соответствующих объектов:
game - класс логической игры
players - логика игрока-компьютера
pieces - классы логических фигур

В файлe start_gui находятся объекты связанные с GUI:
QtPiece, QtCell и QtBoard

В папке pieces_img находятся изображения фигур 
В папке saved_sessions находятся сессии, которые можно загрузить и играть с конкретного места
