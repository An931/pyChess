Игра Шахматы

Запуск через start_gui
Рекомендации по игре: в поле нужно ввести ход, который хотите сделать
формат: координаты фигуры и координаты клетки, на которую она должна перейти
например: "a2 a4" 

В файлах game, players, board, pieces находится логика соответствующих объектов:
game - класс игры
players - класс игрока, а также логика игрока-компьютера
board - класс доски 
pieces - классы фигур
